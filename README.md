# Portifólio - Básico 
Esse é um portifólio simples desenvolvido utilizando apenas HTML e CSS. Com auxílio da plataforma Alura e suas aulas, pude aprender mais sobre como funciona a disponibilidade das tags, links, imagens e organização de arquivos. 

Com o curso de Engenharia de Software na FIAP, pude juntar meus conhecimentos acadêmicos para melhorar a organização do código visando as boas práticas de programação.
