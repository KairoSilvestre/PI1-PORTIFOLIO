# Portifólio - Básico 
Esse é um portifólio simples desenvolvido utilizando apenas HTML e CSS. Com auxílio da plataforma Alura e suas aulas, tive a oportunidade de aprender mais sobre o funcionamento da disponibilidade das tags, links, imagens e organização de arquivos. 

Com o curso de Engenharia de Software na FIAP, juntei meus conhecimentos acadêmicos para melhorar a organização do código visando as boas práticas de programação.
